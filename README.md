☕ Café Billing System
📌 Project Overview

The Café Billing System is a console-based application developed using Python to streamline café operations such as order management, billing, and sales tracking. The system enables users to efficiently handle menu items, process customer orders, and generate accurate bills with automated calculations.

🎯 Features
🧾 Order Management
Add, update, and delete customer orders
Handle multiple items per order
📋 Menu Management (CRUD Operations)
Create new menu items
Read/view available items
Update item details (price, name)
Delete menu items
💰 Automated Billing System
Calculates total cost instantly
Generates detailed bill for each order
📊 Sales Tracking
Stores and retrieves sales data
Helps in basic analysis of orders
⚠️ Exception Handling
Handles invalid inputs gracefully
Prevents application crashes
📁 File Handling & CSV Processing
Stores data locally using files/CSV
Ensures lightweight data management
🗄️ Database Integration
Uses MySQL for persistent data storage
Performs database operations using PyMySQL

🛠️ Technologies Used
Python
Control Statements
Object-Oriented Programming (OOP)
File Handling
Exception Handling
CSV Processing
MySQL – Database management
PyMySQL – Database connectivity
VS Code – Development environment
